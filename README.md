# Blog

## Sobre
```
Blog pessoal com sistema simples de comentarios.
```
## Quickstart

1. Clone o projeto

   ```shell
   git clone git@github.com:Fabio661/FK-Ocean-Blog.git
   ```

2. Rode o projeto na sua maquina:

   ```shell
   python manage.py migrate
   python manage.py runserver
   ```
